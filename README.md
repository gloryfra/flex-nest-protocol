# 🪶 Flex Nest Protocol

**Flex Nest** is a smart contract-based protocol for managing reservations and allocations in decentralized coworking environments. Built using the [Clarity](https://docs.stacks.co/docs/clarity/overview/) language for the [Stacks blockchain](https://stacks.co/), Flex Nest facilitates a peer-to-peer system where users can reserve, trade, and cancel coworking time slots using STX tokens.

---

## 🚀 Features

- 🧾 **Reserve Time Slots**: Users can acquire coworking slots directly with STX tokens.
- 🔁 **Transfer or Share**: Time slots can be transferred or shared between users with built-in allocation limits.
- 📤 **List Unused Slots**: Users can list unused slots at custom prices for others to reserve.
- 💸 **Partial Refunds**: Users receive partial refunds for canceled slots, based on configurable reimbursement rates.
- 🛠 **Admin Controls**: Protocol parameters like fee rates and capacity can be adjusted by the admin.

---

## 🛠 Contract Overview

- **Token Used**: STX
- **Admin Principal**: Defined at deployment
- **Economic Parameters**:
  - Slot Rate: Configurable
  - Protocol Fee: Default 5%
  - Cancellation Reimbursement: Default 90%
- **Time Slot Management**:
  - Acquire, transfer, reallocate, share
  - List slots with pricing
  - Reserve listed slots from others
